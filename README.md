# 滑舌検査アプリ for Pythonista

iPhone の Pythonista で動かすことを想定した、簡易滑舌検査アプリです。

NHK の番組をきっかけに、「1秒間にどれだけ『か』と言えるか」を自分でも試してみたくなり、AI と対話しながら作成しました。

## 主な機能

- 発音文字の選択（か・た・ぱ）
- 測定時間の選択
- 3秒カウントダウン
- 手動カウント
- 録音
- 録音再生
- 音声からの簡易回数推定
- matplotlib による解析グラフ表示
- 横スクロールでの拡大グラフ表示

## 動作環境

- iPhone
- Pythonista

## 使い方

1. `katsuzetsu_check.py` を Pythonista に保存します
2. 実行します
3. 発音文字と測定時間を選びます
4. 「開始」を押します
5. 測定後、必要に応じて録音再生・解析・グラフ表示を行います

## iPhone でコードを取り込む方法

GitHub 上の `katsuzetsu_check.py` を開き、コード全文をコピーして、Pythonista の新規ファイルに貼り付けて保存してください。

## 注意

- このプログラムは個人の学習・試用のための簡易ツールです
- 医療機器や医療判断を目的としたものではありません
- 録音機能を使うため、マイク権限が必要です
- Pythonista の環境によっては matplotlib 関連の表示に差が出る可能性があります
